# Music Delight MOE Calendar v1.1

## Added
- Admin logout button with Supabase sign-out and redirect to login.
- Mobile navigation with hamburger menu.
- Responsive admin layout for phones and tablets.
- Compact expandable Quick Actions menu.

## Fixed
- Large fixed action buttons no longer cover Conflict Center, Teacher Availability, or Teacher Management content.
- Removed the desktop-only minimum page width.
- Added bottom spacing and mobile-safe scrolling.
- Calendar and week views scroll horizontally on small screens instead of being crushed.
- Forms, cards, statistics and modal layouts stack properly on mobile.

## Deployment
Push these files to the GitHub repository connected to Vercel. Vercel will automatically build and deploy the update.
