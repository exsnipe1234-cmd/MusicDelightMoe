# MOE Calendar v1.2

- More versatile PDF importer
  - Supports calendars with 5, 6, or 7 visible weekday columns
  - Uses weekday headings and date positions instead of requiring one exact grid
  - Supports multi-page PDFs
  - More tolerant row and text alignment detection
  - Clearer error when the PDF is scanned or contains no readable lessons
- Conflict Center layout fix
  - Overlapping conflict groups now use the full available width
  - Nested lesson cards shrink correctly instead of spilling over adjacent cards
  - Conflict pairs stack vertically on narrower screens
