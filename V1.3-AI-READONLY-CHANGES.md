# V1.3 Calendar AI — Live Read-Only Search

## Added
- Secure server-side Supabase access for Calendar AI.
- Read-only `search_lessons` tool supporting teacher, school, class, date, date range, time range, unavailable status and instrument inference.
- Read-only `list_teachers` tool.
- Teacher and school name normalisation with ambiguity handling.
- Singapore timezone handling for relative dates.
- Updated assistant UI text and suggested prompts.

## Safety
- The assistant cannot insert, update or delete database records.
- Ambiguous teacher and school names require clarification instead of silent guessing.
- Instrument is currently inferred from `class_name` and related lesson text because there is no dedicated `instrument` column.
